# 孟德尔随机化方法论

## 什么是孟德尔随机化？

孟德尔随机化 (Mendelian Randomization, MR) 是一种利用遗传变异作为工具变量来推断暴露与结局之间因果关系的方法。

## 核心假设

MR 分析依赖于三个核心假设：

1. **关联性假设**: 工具变量（SNPs）与暴露因素强相关
2. **独立性假设**: 工具变量与混杂因素无关
3. **排他性假设**: 工具变量仅通过暴露因素影响结局

## 分析方法

本研究采用多种 MR 方法以确保结果稳健：

| 方法 | 缩写 | 特点 |
|------|------|------|
| Inverse Variance Weighted | IVW | 主要分析方法，假设所有SNPs有效 |
| MR-Egger | Egger | 可检测和校正多效性 |
| Weighted Median | WME | 允许50%的SNPs无效 |
| Weighted Mode | WMO | 假设最常见的效应估计为真 |

## 敏感性分析

- **异质性检验**: 评估不同SNPs效应的异质性
- **多效性检验**: MR-Egger截距检验水平多效性
- **Leave-one-out**: 评估单个SNP对结果的影响

## 结果解读

- OR < 1: 暴露对结局有保护作用
- OR > 1: 暴露增加结局风险
- P < 0.05: 结果具有统计学意义
